# Gaur Commerce Classes (Next.js Edition)

This is a modern web application built with **Next.js 14**, **React**, **TypeScript**, and **Tailwind CSS**.

## Features
- **Modern Architecture**: Built on React Server Components.
- **Performance**: Static generation and server-side rendering.
- **Styling**: Tailwind CSS for responsive, professional design.
- **Interactivity**: Client-side filtering and form handling.
- **Icons**: Lucide React icons.

## Prerequisites
- Node.js (v18 or higher)

## How to Run Locally
1. Open the terminal in this folder.
2. Install dependencies (if not already done):
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## How to Deploy
This app is optimized for Vercel.
1. Push to GitHub.
2. Import project in Vercel.
3. It will auto-detect Next.js and deploy.
